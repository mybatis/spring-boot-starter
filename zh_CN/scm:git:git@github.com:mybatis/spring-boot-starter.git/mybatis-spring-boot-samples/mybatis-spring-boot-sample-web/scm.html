匿名访问

可以使用此命令从 Git 匿名签出源代码(参见 https://git-scm.com/docs/git-clone):

只有项目开发人员能够通过这一方法访问Git项目树。 (参见 https://git-scm.com/docs/git-clone).

$ git clone git@github.com:mybatis/spring-boot-starter.git
